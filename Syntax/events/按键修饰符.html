系统修饰键 2.1.0 新增
.ctrl
.alt
.shift
.meta
在 Mac 系统键盘上，meta 对应 command 键 (⌘)。在 Windows 系统键盘 meta 对应 Windows 徽标键 (⊞)。

.exact 修饰符 2.5.0 新增
.exact 修饰符允许你控制由精确的系统修饰符组合触发的事件。
@click.ctrl 即使 Alt 或 Shift 被一同按下时也会触发
@click.ctrl.exact 有且只有 Ctrl 被按下的时候才触发
@click.exact 没有任何系统修饰符被按下的时候才触发
